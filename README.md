## Development

Set the required environment variables in your `.env.local`:

```
NEXT_PUBLIC_BASE_URL=http://localhost:3000
NEXT_PUBLIC_API_URL=http://localhost:3000/api
```

Install the dependencies and run the app:

```shell
pnpm i
pnpm run dev
```
